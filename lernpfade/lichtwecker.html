Projekt-Lernpfad: Dein eigener Lichtwecker

Schritt-für-Schritt von der Simulation bis zur gedruckten Box. Perfekt für Einsteigerinnen und Einsteiger ohne Vorkenntnisse.

← Zurück zur Jugend-forscht-Übersicht
Was ist ein Lichtwecker? Ein Lichtwecker simuliert einen natürlichen Sonnenaufgang. Statt von einem lauten Ton wirst du sanft durch schrittweise heller werdende LEDs geweckt. In diesem Lernpfad bauen wir ein Modell mit einem Arduino Uno, LEDs, einem Taster zum Ausschalten und einer maßgeschneiderten 3D-gedruckten Hülle!
Schritt 1

Virtuelles Testen mit Tinkercad Circuits

Bevor wir echte Bauteile anfassen, simulieren wir die gesamte Schaltung kostenlos im Browser. Das verhindert Hardware-Fehler und lässt uns den Code sicher testen.

Ablauf in Tinkercad:

  1. Gehe auf tinkercad.com, erstelle ein kostenloses Konto und klicke im linken Menü auf "Designs" und dann auf "Circuits" (Schaltungen).
  2. Klicke auf "Neu erstellen" → "Schaltung".
  3. Ziehe folgende Komponenten aus der rechten Leiste per Drag-and-Drop auf die Arbeitsfläche:
    • Arduino Uno R3
    • Breadboard Small (Steckplatine)
    • Mehrere LEDs (z. B. weiß oder gelb)
    • Widerstände (330 Ohm für die LEDs, um sie vor dem Durchbrennen zu schützen)
    • 1x Drucktaster (Pushbutton) & 1x 10k Ohm Widerstand (als Pull-Down-Widerstand)
  4. Verdrahte die Bauteile: Verbinde die Kathoden (kurzes Bein) der LEDs mit der Minus-Schiene (GND) und die Anoden über einen Widerstand mit den digitalen Pins des Arduinos (z.B. Pin 9, 10, 11).
Tipp für Einsteiger: Tinkercad zeigt dir direkt an, wenn eine LED zu viel Strom bekommt. Achte darauf, dass die Widerstände richtig auf "Ohm" (Ω) eingestellt sind, nicht auf "Kilo-Ohm" (kΩ) für die LEDs!
Schritt 2

Das Gehäuse designen in Tinkercad 3D

Damit der Wecker auf dem Nachttisch gut aussieht, entwerfen wir eine passgenaue Box mit abnehmbarem Deckel.

Ablauf im 3D-Editor:

  1. Wechsle in Tinkercad zu "3D-Designs" und erstelle ein neues Design.
  2. Ziehe einen Quader auf die Arbeitsfläche und skaliere ihn auf die gewünschte Größe (z. B. Breite 120mm, Tiefe 80mm, Höhe 70mm).
  3. Nutze die Form "Hohlkörper" (ein leicht kleinerer Quader), um die Box von oben auszuhöhlen, sodass eine Wandstärke von ca. 2 bis 3mm stehen bleibt.
  4. Vergiss die Aussparungen nicht! Platziere kleine Hohlkörper-Zylinder oder -Quader an den Seiten, um Löcher für das Stromkabel (USB oder Strombuchse) und den Wecker-Ausschaltknopf zu schneiden.
  5. Der Deckel: Erstelle eine flache Platte, die exakt auf die Box passt. Du kannst Text-Formen (wie "Lichtwecker" oder ein Power-Symbol) hinzufügen, die entweder erhaben herausstehen oder als Hohlkörper hineingeschnitten werden, damit das Licht durchscheint.
  6. Exportiere deine fertigen Formen separat als .STL-Datei für den 3D-Drucker.
Schritt 3

Die Arduino IDE vorbereiten

Um dem echten Arduino Leben einzuhauchen, nutzen wir die offizielle Programmierumgebung (IDE).

Software einrichten:

  1. Lade die kostenlose Arduino IDE von der offiziellen Website (arduino.cc) herunter und installiere sie.
  2. Verbinde deinen echten Arduino Uno über das blaue USB-Kabel mit deinem Computer.
  3. Öffne die Arduino IDE und stelle die Verbindung her:
    • Gehe auf Werkzeuge → Board und wähle "Arduino Uno" aus.
    • Gehe auf Werkzeuge → Port und wähle den COM-Port aus, an dem dein Arduino erkannt wurde (z.B. COM3 oder COM4).
Schritt 4

Der Programmcode (Sketch)

Kopiere diesen Einsteiger-Code in deine Arduino-IDE. Er lässt die LEDs langsam nacheinander hochdimmen (Sonnenaufgang) und wartet darauf, dass der Taster gedrückt wird, um das Licht wieder auszuschalten.

// DIY Lichtwecker - Basiscode für Einsteiger const int ledPins[] = {9, 10, 11}; // Die Pins, an denen die LEDs hängen const int anzLeds = 3; const int buttonPin = 2; // Pin für den Ausschalter void setup() { for(int i = 0; i < anzLeds; i++) { pinMode(ledPins[i], OUTPUT); // LEDs als Ausgang definieren } pinMode(buttonPin, INPUT); // Taster als Eingang definieren } void loop() { // 1. Der "Sonnenaufgang" startet: LEDs nacheinander hochdimmen for(int i = 0; i < anzLeds; i++) { for(int helligkeit = 0; helligkeit <= 255; helligkeit++) { analogWrite(ledPins[i], helligkeit); // PWM-Signal zur Helligkeitssteuerung delay(20); // Geschwindigkeit des Dimmens (für echten Wecker stark erhöhen!) // Zwischendurch prüfen, ob der Taster gedrückt wurde if(digitalRead(buttonPin) == HIGH) { ausschalten(); return; } } } // 2. Warten, bis der Wecker durch Tastendruck beendet wird while(true) { if(digitalRead(buttonPin) == HIGH) { ausschalten(); break; } delay(50); } } void ausschalten() { for(int i = 0; i < anzLeds; i++) { digitalWrite(ledPins[i], LOW); // Alle LEDs aus } delay(5000); // 5 Sekunden Pause, bevor der Loop von vorn starten könnte }

Klicke in der Arduino IDE oben links auf das Häkchen (Überprüfen), um den Code zu testen, und danach auf den Pfeil nach rechts (Hochladen), um ihn auf das Board zu spielen.

Schritt 5

Zusammenbau & Echttest

Sobald der Code aufgespielt ist, kannst du den Arduino vom PC trennen. Im finalen Gehäuse kann der Wecker autark über ein USB-Netzteil oder eine 9V-Blockbatterie betrieben werden.
