🔌 Schaltung bauen & programmieren

I

Neuen Schaltkreis erstellen

Erstellt bei Tinkercad einen neuen Schaltkreis mit eurem Klassencode – wie im 3D-Drucker-Lernpfad beschrieben.

II

Ampelschaltung aufbauen

Konstruiert eine Ampelschaltung mit drei LED-Lampen (rot, gelb, grün) und jeweils einem 220-Ω-Schutzwiderstand.

III

Programmierung planen

  • Klickt auf die Schaltfläche „Code".
  • Zieht die Wiederholschleife „forever" in das Programmierfeld.
  • Setzt den Pin zu einer LED über den Ausgangsblock auf „high" – die LED soll leuchten.
  • Gebt eine Wartezeit ein, die das Leuchten in diesem Zustand hält.
  • Testet euer Programm über „Simulation starten" oben rechts.
  • Wenn ihr zufrieden seid: Schaltung und C++-Code lassen sich jederzeit exportieren.
// vereinfachtes Ablaufprinzip
forever {
  digitalWrite(rotPin, HIGH); // Rot an
  wait(3000);
  digitalWrite(gelbPin, HIGH); // Rot-Gelb
  wait(1000);
  digitalWrite(rotPin, LOW);
  digitalWrite(gelbPin, LOW);
  digitalWrite(gruenPin, HIGH); // Grün an
  wait(3000);
}
IV

Code auf den echten Arduino übertragen

  • Kopiert den Code, der aus eurem Blockcode-Programm übersetzt wurde.
  • Startet die Arduino IDE.
  • Verbindet den Arduino mit dem Rechner und wählt ihn in der IDE aus.
  • Fügt den kopierten Code ein, schließt den Arduino an und ladet den Code hoch.

🧠 Vertiefung: Gefährliche Kreuzung sicher programmieren

Bei einer echten Kreuzung reicht eine einzelne Ampel nicht: Fahrzeuge und Fußgänger:innen müssen koordiniert werden. Dafür lohnt sich ein Blick auf die objektorientierte Programmierung (OOP) – Autoampel und Fußgänger:in werden dabei als eigenständige Objekte mit Zuständen und Verhalten modelliert.

🚗 Objekt: Autoampel

  • Zustand: Rot / Rot-Gelb / Grün / Gelb
  • Methode: Phase wechseln
  • Methode: Aktuellen Zustand anzeigen

🚶 Objekt: Fußgänger:in

  • Zustand: Warten / Gehen
  • Methode: Straße überqueren
  • Bedingung: nur bei Grün für Fußgänger:in
🔗
Nutzt inf-schule.de/oop/scratch, um das Konzept der objektorientierten Programmierung am Beispiel der gefährlichen Kreuzung besser zu verstehen. Löst schrittweise bis einschließlich Abschnitt 2.3 („Übungen") – das legt die Grundlage, um später komplexe Verkehrssteuerungen selbst zu planen.

📚 Sachkompetenz-Check

Eine LED hat zwei Beine unterschiedlicher Länge. Welches Bein ist der Pluspol (Anode)?
Das längere Bein. Es wird mit dem stromführenden Pin verbunden.
Tipp: Kurz vor Minus, lang vor Plus.
Welche Funktion hat der 220-Ohm-Widerstand in einem einfachen LED-Stromkreis mit einem Arduino?
Er schützt die LED vor zu hohem Strom.
Der Widerstand begrenzt den Stromfluss vom Arduino-Pin zur LED, um eine Beschädigung zu verhindern.
Welchen Wert in Ohm (Ω) hat der Schutzwiderstand, der typischerweise für eine Standard-LED am 5V-Arduino-Pin verwendet wird?
220 Ω ist ein gebräuchlicher und sicherer Wert, um den Strom für die meisten LEDs an einem 5V-Ausgang zu begrenzen.
Womit wird der Minuspol (Kathode) einer LED in einem Arduino-Schaltkreis typischerweise verbunden?
Mit dem GND-Pin.
GND steht für „Ground" (Masse) und ist der gemeinsame Minuspol für den Stromkreis.
Welches Bauteil wird verwendet, um elektronische Komponenten wie LEDs und Widerstände ohne Löten miteinander zu verbinden?
Eine Steckplatine (Breadboard).
Speziell dafür konzipiert, Prototypen von Schaltungen schnell und reversibel aufzubauen.
Vervollständige den Satz: In der Arduino IDE wird der ___ geschrieben, um die Lichtsignalabfolge der Ampel zu steuern.
Code – der Programmcode enthält die Befehle, die dem Arduino sagen, wann welche LED leuchten soll.
Eine rote LED in eurer Schaltung leuchtet nicht. Das Kabel steckt in Pin 9, der Code ist korrekt. Was ist wahrscheinlich das Problem?
Die rote LED ist verpolt eingesetzt.
Bringt die Farben einer deutschen Fußgängerampel in die richtige Reihenfolge, beginnend mit „Halt".
Rot, Rot-Gelb, Grün, Gelb.
Stehenbleiben bei Rot, Gehen bei Grün.
Der Arduino und die Ampelschaltung werden mit Gleichstrom betrieben. Der Begriff „Gleichstrom" bedeutet, dass der Strom …
… nur in eine Richtung fließt.
Gleichstrom (DC) fließt konstant vom Plus- zum Minuspol, im Gegensatz zu Wechselstrom (AC).

Woher hat der Arduino-Mikrocontroller seinen Namen?
Von einer Bar in Ivrea (Italien), in der sich die Entwickler trafen.
Nicht zu verwechseln mit Arduin von Ivrea, einem italienischen König – der lieferte nur den Namen der Bar.
Die erste elektrische Ampelanlage der Welt wurde 1914 in Betrieb genommen. In welcher Stadt war das?
Cleveland, USA.
Am 5. August 1914 wurde in Cleveland, Ohio, die erste als sicher geltende elektrische Ampelanlage installiert.
Der berühmte Ampelmann, eine Symbolfigur auf vielen Fußgängerampeln, stammt ursprünglich aus …
… der DDR (Ost-Berlin), 1961.
Der Verkehrspsychologe Karl Peglau entwarf den Ampelmann 1961 in Ost-Berlin, um die Sicherheit für Fußgänger:innen zu erhöhen.
