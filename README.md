# Jugend forscht – Forschungs-AG

Webseite der Jugend-forscht-/Forschungs-AG des Rosa-Luxemburg-Gymnasiums.
Live unter: https://nitrogen-ai.github.io/Jugend-Forscht/

Ausgelagert aus [nitrogen-ai/Informatik](https://github.com/nitrogen-ai/Informatik), damit die AG-Seite unabhängig
von der Unterrichtsmaterial-Verwaltung wachsen kann.

## Struktur

```
index.html          Übersichtsseite (Hero, Lernpfade, Ergebnisse, Experimente, Wettbewerbe)
lernpfade/           Interaktive Schritt-für-Schritt-Lernpfade
ergebnisse/           Aufbereitete Forschungsergebnisse
experimente/          (folgt) Lernpfade zu unseren Experimentierstationen
assets/bilder/        Bilder, Logos
```

Jede Seite ist eine einzelne, offline lauffähige HTML-Datei ohne CDN-Abhängigkeiten
(Grafiken sind als Vektorgrafiken oder komprimierte Bilder direkt eingebunden).

## Öffentlich — Vorsicht

Dieses Repo ist public. **Keine Schüler-Klarnamen, Noten, Fotos von Personen oder
sonstigen personenbezogenen bzw. vertraulichen Daten** gehören in dieses Repo — weder
in Dateien noch in Commit-Nachrichten. Rohmaterial (Dokumente, unbearbeitete Fotos,
Empfehlungsschreiben) bleibt lokal und wird nicht veröffentlicht; auf dieser Seite
erscheinen nur redaktionell aufbereitete, anonymisierte Inhalte.
